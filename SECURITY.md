## Bug Reports

If you find a bug, please [open an issue](https://github.com/qkal/superproxy/issues) with:

1. **Superproxy version** (`superproxy --version`)
2. **Bun version** (`bun --version`)
3. **OS and architecture**
4. **Steps to reproduce**
5. **Expected vs actual behavior**
6. **Relevant logs** (run with `--log-level debug`)

## Security Vulnerabilities

Please **do not** report security vulnerabilities in public issues. Email the maintainers directly or use [GitHub's security advisory feature](https://github.com/qkal/superproxy/security/advisories).
